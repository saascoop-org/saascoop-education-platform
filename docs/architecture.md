# Architecture

## Initial architecture

The first version is a static documentation platform powered by GitHub, MkDocs Material, GitHub Actions and GitHub Pages.

## Components

- GitHub repository: source of truth.
- MkDocs Material: documentation site generator.
- GitHub Pages: hosting.
- GitHub Actions: build and deployment automation.
- Course repositories: independent learning assets.

## Future evolution

- Course catalog.
- Shared templates.
- GitHub login.
- Learner profile.
- Progress tracking.
- XP and badges.
