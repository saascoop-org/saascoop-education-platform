# SaaSCoop Education Platform

Open source education platform for SaaSCoop courses, learning paths, community onboarding and multilingual technical education.

## Goals

- Publish open multilingual courses.
- Support SaaSCoop onboarding.
- Reuse a standard course structure.
- Keep the platform low-cost, maintainable and community-driven.

## Initial courses

- GitHub for Beginners
- Microsoft Fabric for Beginners
- AI Engineering Foundations
